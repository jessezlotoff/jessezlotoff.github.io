Tab Functions

by: Jesse Zlotoff

Although everything is potentially possible in R, not everything is easy. In particular, there doesn’t seem to be an equivalent of the Stata tab function that I’ve found extremely useful for analysis. I created a new function to provide the same features to the questionnaire library and have now made it available to others on Github.

The main tab function is a combination of Stata’s tab and svy: tab functions, creating weighted or unweighted tabs. You can read the full vignette here.

Planned Features
I hope to expand on the existing functionality to:

  • Collapse multiple categories into new ones
  • Better format the tab outputs for sharing
  • Create toplines documents with multiple row and column variables
  • Create bar (and other) charts based on the results
